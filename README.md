# Money Manager (GitHub Pages, Single User)

✅ Works on GitHub Pages (no server)  
✅ Single user only (no sharing)  
✅ Data won't disappear when you update code (stored in browser IndexedDB)  
✅ Manual backup/export + restore/import (JSON)

## Deploy (one-time)
1. Push this repo to GitHub (main branch)
2. Go to **Settings → Pages → Source: GitHub Actions**
3. Open: https://phyuphuycho.github.io/money-manager/

## Note about data
- Data lives inside your browser (IndexedDB).
- If you clear site data / browser storage, it can be lost.
- Use **Backup JSON** button sometimes.
